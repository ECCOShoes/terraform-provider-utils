# utils Provider

The `utils` provider exposes utility functions for common data manipulation tasks in Terraform configurations. It does not manage any infrastructure resources.

## Functions

| Function | Description |
|----------|-------------|
| [`deepmerge`](functions/deepmerge.md) | Deep-merge multiple objects into one |
| [`readsops`](functions/readsops.md) | Decrypt SOPS-encrypted content |
| [`jsonexpand`](functions/jsonexpand.md) | Expand a flat map into a nested object |
| [`jsonflatten`](functions/jsonflatten.md) | Flatten a nested object into a flat map |

## Example Usage

```terraform
provider "utils" {}
```
